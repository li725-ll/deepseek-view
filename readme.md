# Deepseek-view

This is a deepseek client and you need to have the deepseek key to use it. You just need to set the key to talk to the ai.

> The projects created at random are rough

## Setup

Installing dependencies

```shell
pnpm install
```

Running the script

```shell
pnpm run start
```

## Usage

1. Set key
2. Selection model
3. chat
